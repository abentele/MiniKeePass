Welcome.

MiniKeePassX is a brand new password tool for Mac OS X. While it is based on the KeePass library implemented in the MiniKeePass iOS App, the application is inspired by the reference implementation KeePass Password Save and KeePassX. MiniKeePassX can load and save files created with KeePass Password Save, MiniKeePass or KeePassX.

Schedule

MiniKeePassX is in an early development stage. I use my spare time to implement it, so don't ask me when it will be ready, but maybe you will see the first distributable version within the first quarter of 2013.

MiniKeePassX will be distributed over the App Store for OS X.

Why implementing another KeePass tool

I was using KeePassX for a few years. KeePassX is another very good KeePass tool with an emphasis on platform independence. The implementation is based on C++ and the QT library. The downsides of this platform independent approach is poor usability and missing OS X specific features like versions and other document based features. With KeePassX 2 the look and feel of the application will be better, but I believe devoutly that a native OS X application can provide a much better user experience.

Design principles

As a contributor you should be aware of these design principles:

The github repository is a fork of the MiniKeePass repository. The implementation is made in a branch named "osx".

All OS X related implementation is an extension to the original MiniKeePass code. The original MiniKeePass code should not be changed for the following reasons:

  • the MiniKeePass code will change in future. It would be easier to merge improvements in the MiniKeePass code if we don't change that code.
  • maybe the creators of the original MiniKeePass app will decide one day to merge the OS X related code to their repository (I would be very happy about that!). This will be simpler if there are no merge conflicts. My vision is to have one repository for both MiniKeePass for iOS and OS X one day.
